# Obter Imagens

Alguns trabalhos não possuem disponibilidade de todas as imagens utilizadas nos casos de teste.

Para isso, foi utilizado este script, que coloca as imagens em uma mensagem de e-mail e envia para um destinatário qualquer.

Como o script será submetido no website para poder obter as imagens, recomenda-se o uso de uma conta de e-mail temporária ou pública, visto que o usuário e senha poderão ficar armazenados no website.
